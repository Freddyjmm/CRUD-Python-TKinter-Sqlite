    CREATE READ UPDATE DELETE CRUD

    Un CRUD es un software que permite crear, leer, actualizar y borrar información, para este proyecto se uso Tkinter para crear la interfaz de aplicación de escritorio y sqlite como software para persistir los datos

    Código

    Este software se desarrollo pensando en la problemática de llevar el control de las fechas de vencimiento de productos que se venden en una tienda

    TKinter

    TKinter es un framework de Python para desarrollar aplicaciones de escritorio el cual se puede ejecutar en cualquier sistema operativo que tenga instalado el interprete de python

    Sqlite

    Es un módulo incluido en las librerías estándar de python desde la versión 3.X que permite crear un base de datos a partir de archivos estáticos, diferenciándola del habitual modelo cliente servidor

    Este Proyecto

    En este proyecto se combina las virtudes del framework TKinter y la integración de la base de datos que provee para crear un interfaz que divide las funciones del programa en tres vistas diferentes según verbos que establece un crud (crear, leer, actualizar y borrar)
